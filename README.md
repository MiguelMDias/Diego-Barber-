💈 Diego Barber

Site moderno e responsivo desenvolvido para uma barbearia premium, com foco em design sofisticado, experiência do usuário e performance.

✨ Sobre o projeto

O **Diego Barber** é um site institucional que apresenta serviços, planos, equipe e localização de uma barbearia fictícia. O projeto foi desenvolvido com o objetivo de praticar e demonstrar habilidades em desenvolvimento front-end, criando uma interface elegante e interativa.

🚀 Tecnologias utilizadas

- HTML5
- CSS3 (com variáveis e design responsivo)
- JavaScript (interações e animações)
- Google Fonts

🎯 Funcionalidades

- ✅ Layout totalmente responsivo
- ✅ Menu mobile com hamburger
- ✅ Animações suaves ao rolar a página (scroll reveal)
- ✅ Navegação com scroll suave
- ✅ Seções organizadas (Sobre, Serviços, Planos, Equipe, Clientes e Localização)
- ✅ Design moderno com foco em UX/UI

📱 Responsividade

O site foi desenvolvido para funcionar perfeitamente em:
- 📱 Dispositivos móveis
- 💻 Tablets
- 🖥️ Desktops

🎨 Destaques do projeto

- Identidade visual premium (preto + dourado)
- Uso de tipografia moderna e elegante
- Animações que melhoram a experiência do usuário
- Estrutura de código organizada e separada (HTML, CSS e JS)

📍 Objetivo

Este projeto faz parte do meu portfólio como desenvolvedor front-end, com o objetivo de demonstrar minhas habilidades na criação de interfaces modernas e funcionais.
